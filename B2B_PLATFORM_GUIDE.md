### 📍 第一步：LinkedIn 公司主页（最重要！免费，流量大）

**为什么重要**：LinkedIn 是全球最大的职业社交平台，3亿+用户。很多海外采购商会在 LinkedIn 上搜索供应商。

#### 注册步骤：

1. **先创建个人账号**
   - 打开 https://www.linkedin.com/signup
   - 用邮箱注册个人账号（建议用 lily@tzcypress.com）
   - 填写个人信息：
     - Name: Lily（你的名字）
     - Title: Sales Manager at Taizhou Cypress Import & Export Co., Ltd.

2. **创建公司主页**
   - 登录后访问：https://www.linkedin.com/company/setup/create/
   - 填写以下信息：

| 字段 | 填写内容 |
|------|---------|
| Page Name* | Taizhou Cypress / Cypress Shoes |
| Page URL（建议） | cypress-shoes 或 taizhou-cypress |
| Industry | Apparel & Fashion / Manufacturing |
| Company Size | 11-50 employees |
| Website | https://www.cypressfootwear.com |
| Tagline | Premium Sport Shoes Manufactured for the World |
| Description | 复制上面的「版本A：简短版」 |
| Logo | 上传 CYPRESS LOGO 图片 |
| Cover Image | 上传工厂/样品室图片 |

3. **完善主页**
   - 添加公司 specialties（专长）：`Sport Shoes, OEM/ODM Manufacturing, Kids Shoes, Men's Shoes, Women's Shoes, Footwear Export`
   - 发布第一篇帖子（介绍公司和网站）
   - 在 About 里加上独立站链接

4. **关键优化**
   - 个人 Profile 里把公司网站填上
   - 连接行业相关的客户和同行
   - 定期发帖（产品更新、展会动态等）

## 七、注意事项和避坑提醒

1. **所有平台用同一个公司名和联系方式** — 方便客户跨平台找到你
2. **邮箱用域名邮箱 `lily@tzcypress.com`** — 比 QQ/163 邮箱更专业
3. **WhatsApp 号码一定要加** — 海外客户最常用的沟通工具
4. **每个平台的描述不要完全一样** — 改动一些措辞，避免被判重复内容
5. **定期登录活跃** — 长期不登录会降低排名
6. **不要在 DIYTrade 加外链** — 会直接封号
7. **营业执照准备好电子版** — 大部分平台需要企业认证
